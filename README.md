## 📌 Опис

Проєкт містить два застосунки типу **«Форма опитування»**, реалізовані за допомогою двох різних UI-фреймворків:

| Проєкт | Технологія | Мова | Колір теми |
|--------|-----------|------|------------|
| `WinForms` | Windows Forms | C# | 🔵 Синій |
| `WPF` | Windows Forms | Visual Basic .NET | 🔴 Червоний |

Обидва застосунки запитують у користувача три відповіді, валідують введені дані та зберігають результат у текстовий файл.

---

## 🖥️ Функціональність

- **3 поля вводу:**
  - `Name` — ім'я користувача
  - `Age` — вік (ціле число від 1 до 120)
  - `Favorite Language` — вибір мови програмування через ComboBox з автодоповненням

- **Валідація даних** перед збереженням:
  - Поле `Name` не може бути порожнім
  - Поле `Age` приймає лише цілі числа у діапазоні 1–120
  - Поле `Favorite Language` не може бути порожнім

- **Збереження у файл** `survey_answers.txt` у директорії запуску застосунку

- **Повідомлення** про успішне збереження або помилку валідації

- **Очищення полів** після успішного збереження

## 🚀 Запуск

### Вимоги
- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- Windows OS
- Visual Studio 2022+ або Visual Studio 2026

### Через Visual Studio
1. Відкрити `LAB1_UI.slnx`
2. Обрати стартовий проєкт (`WinForms` або `WPF`)
3. Натиснути `F5`

### Через термінал
```powershell
# WinForms (C#)
cd WinForms
dotnet run

# WPF (VB.NET)
cd WPF
dotnet run
```

## 🎨 UI

### WinForms (C#) — синя тема
- Заголовок: `Color.FromArgb(37, 99, 235)` — blue-600
- Кнопка Submit: та ж синя кольорова схема
- Підпис внизу: *WinForms*

### WPF (VB.NET) — червона тема
- Заголовок: `Color.FromArgb(220, 38, 38)` — red-600
- Кнопка Submit: та ж червона кольорова схема
- Підпис внизу: *WPF*

Обидві форми використовують шрифт **Segoe UI**, фіксований розмір вікна `460×320` та відцентроване розміщення на екрані.
